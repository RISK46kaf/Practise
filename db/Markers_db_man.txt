Добавление:

Соединитель маркера/пациента
`Connector_entry` (in URL TEXT, in PatId int, in Ename text)
URL - путь к картинке
PatId - id пациента
Ename - название/описание записи

Добавлениие маркера
`Fill_FMarker` (in mName text, in color int, in thickness int, in scale int, in inURL text, in x int, in y int, in form text)
mName - название (тут по сути может быть что угодно, но лучше что то уникальное)
color - цвет
thickness - толщина линии
scale - масштаб
URL - адрес. К чему конкретно не важно, но по нему определяется принадлежность маркера к конкретной картинке.
Если адрес уже есть в базе - маркер привяжется к существующей записи, если нет - сгенерируется новая
x -
y - координаты начальной точки маркера
form - форма маркера



Для последующего добавления точек
`Fill_Point` (in x int, in y int, 0 int)
x
y - координаты новой точки, которая будет относиться к последнему созданному маркеру
последний параметр обязательно должен быть 0;
